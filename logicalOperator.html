Logical Operators in Programming

Understanding the most commonly used logical operators.

Operator Description Syntax Example Result
AND (&&) True if both operands are true A && B true && false false
OR (||) True if at least one operand is true A || B true || false true
NOT (!) Reverses the boolean value !A !true false
Equality (==) True if operands are equal A == B 5 == 5 true
Inequality (!=) True if operands are not equal A != B 5 != 3 true
Greater than (>) True if the left operand is greater than the right A > B 7 > 4 true
Less than (<) True if the left operand is less than the right A < B 3 < 9 true
Greater than or equal (>=) True if the left operand is greater or equal A >= B 5 >= 5 true
Less than or equal (<=) True if the left operand is less or equal A <= B 3 <= 8 true
