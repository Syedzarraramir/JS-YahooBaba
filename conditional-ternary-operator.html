Conditional (Ternary) Operator

The conditional (ternary) operator is a shorthand way to write simple if-else conditions in a single line. It is commonly used when you need to assign a value based on a condition.

Syntax

The syntax for the ternary operator is:

condition ? expression_if_true : expression_if_false;
If the condition is true, the expression before the colon (expression_if_true) is executed. If the condition is false, the expression after the colon (expression_if_false) is executed.

Example

Here’s a simple example that uses the ternary operator to determine whether a person is an adult or a minor based on their age:

let age = 20;
let result = (age >= 18) ? "You are an adult." : "You are a minor.";
console.log(result);
In this example, if age is 18 or greater, the result will be "You are an adult."; otherwise, it will be "You are a minor."

Another Example

Here’s a ternary operator that checks if a number is even or odd:

let number = 5;
let result = (number % 2 === 0) ? "Even" : "Odd";
console.log(result);
This checks whether the number is divisible by 2 (i.e., even). If the condition is true, it returns "Even"; otherwise, it returns "Odd."

When to Use the Ternary Operator

The ternary operator is useful when you need a quick and simple way to return values based on a condition. However, for more complex logic, using regular if-else statements may improve readability.

Conclusion

The conditional (ternary) operator is a great tool for simplifying your code, especially for small, simple conditions. However, it should be used wisely to avoid making the code difficult to read.
