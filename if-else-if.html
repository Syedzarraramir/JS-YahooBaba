If, Else If, and Else Statements in Programming

If Statement

The if statement is used to execute a block of code if a specified condition is true.

Syntax

if (condition) {
    // Code to execute if the condition is true
}

Example

let age = 20;

if (age >= 18) {
    console.log("You are eligible to vote.");
}

Else If Statement

The else if statement is used to specify a new condition to test if the first condition is false. You can have multiple else if conditions.

Syntax

if (condition1) {
    // Code to execute if condition1 is true
} else if (condition2) {
    // Code to execute if condition1 is false and condition2 is true
}

Example

let marks = 75;

if (marks >= 90) {
    console.log("Grade: A");
} else if (marks >= 80) {
    console.log("Grade: B");
} else if (marks >= 70) {
    console.log("Grade: C");
}

Else Statement

The else statement is used to execute code if none of the conditions in the if or else if statements are true.

Syntax

if (condition) {
    // Code to execute if the condition is true
} else {
    // Code to execute if the condition is false
}

Example

let temperature = 35;

if (temperature > 40) {
    console.log("It's too hot!");
} else if (temperature < 10) {
    console.log("It's too cold!");
} else {
    console.log("The weather is moderate.");
}

Flow of Execution

The if-else if-else chain works as follows:

  • The if condition is checked first. If true, the code inside the if block is executed.
  • If the if condition is false, the program moves to the else if block and checks its condition.
  • If all conditions are false, the code inside the else block (if present) is executed.
