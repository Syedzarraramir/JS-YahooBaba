Understanding the If-Else Statement in Programming

The if-else statement allows conditional execution of code blocks based on boolean expressions.

Basic Syntax

if (condition) {
    // Code to execute if the condition is true
} else {
    // Code to execute if the condition is false
}

Example of If-Else Statement

let age = 18;

if (age >= 18) {
    console.log("You are eligible to vote.");
} else {
    console.log("You are not eligible to vote.");
}

Explanation of Example

Condition True Block False Block
age >= 18 "You are eligible to vote." "You are not eligible to vote."

Nested If-Else Statement

If-Else statements can also be nested to check multiple conditions:

let marks = 85;

if (marks >= 90) {
    console.log("Grade: A");
} else if (marks >= 80) {
    console.log("Grade: B");
} else {
    console.log("Grade: C");
}

Ternary Operator (Short Form of If-Else)

The ternary operator provides a shorthand way to write simple if-else conditions:

let age = 20;
let result = (age >= 18) ? "You are an adult." : "You are a minor.";
console.log(result);
