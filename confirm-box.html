JavaScript Confirm Box

The confirm() function in JavaScript displays a dialog box with two buttons: OK and Cancel. It is typically used to ask for user confirmation before proceeding with an action.

Syntax

The syntax of the confirm box is:

let result = confirm("Your message goes here");

if (result) {
    // User clicked OK
} else {
    // User clicked Cancel
}
The confirm() method returns true if the user clicks "OK" and false if the user clicks "Cancel". This allows you to take appropriate actions based on the user's response.

Example

Here’s an example of a confirm box that asks the user if they are sure they want to proceed:

<button onclick="showConfirm()">Click Me</button>

function showConfirm() {
    let result = confirm('Are you sure you want to proceed?');
    if (result) {
        alert('You pressed OK!');
    } else {
        alert('You pressed Cancel!');
    }
}
In this example, when the user clicks the "Click Me" button, a confirmation dialog appears with the message "Are you sure you want to proceed?". If the user clicks "OK", an alert displays "You pressed OK!", and if they click "Cancel", it displays "You pressed Cancel!".

When to Use confirm()

The confirm() function is useful for actions that require user consent, such as deleting data, submitting forms, or navigating away from a page with unsaved changes. It ensures that the user is aware of the action and can cancel if needed.

Conclusion

The confirm() function is an effective way to prompt users for confirmation before executing potentially irreversible actions. It’s a simple and effective method for improving user interaction in your applications.
